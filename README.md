# type-socket

simple terminal-based chatroom using Bun & WebSockets. supports multiple clients on one server.

<div align="center">
  <img src="./static/demo.webp"alt="demo" >
</div>

## requirements
[bun v1.2](https://bun.com/)

## quick install 
```bash
bun install
```

## run the server 
```bash
bun run server
```

## run the client 
```bash
bun run client
```
